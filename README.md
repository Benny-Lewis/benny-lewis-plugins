# benny-lewis-plugins

Claude Code plugin marketplace by Benny Lewis.

## Setup

Add the marketplace (one-time):

```
/plugin marketplace add Benny-Lewis/benny-lewis-plugins
```

## Plugins

| Plugin | Install | Description |
|--------|---------|-------------|
| [home-assistant-assistant](https://github.com/Benny-Lewis/home-assistant-assistant) | `/plugin install home-assistant-assistant@benny-lewis-plugins` | Manage Home Assistant through natural language — automations, scripts, scenes, dashboards, naming, deployment |
| [garden-bot](https://github.com/Benny-Lewis/garden-bot) | `/plugin install garden-bot@benny-lewis-plugins` | Gardening and landscaping assistant — persistent profiles, expert plant advice, visual landscape design |
| [personal-tools](https://github.com/Benny-Lewis/personal-tools) | `/plugin install personal-tools@benny-lewis-plugins` | Personal workflow tools — Notepad++ tab cleanup/triage, and more |
